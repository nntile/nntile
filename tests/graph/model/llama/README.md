# Llama Model Tests

Tests for the NNTile C++ Llama model components. Testing is done via safetensors disk files (no Python wrappers).

RMSNorm is a shared module in `nntile::module` (see `include/nntile/module/rms_norm.hh`) and can be reused by other models (GPT, etc.).

## Layout

All tensors use **(hidden_size, seq, batch)** in Fortran (column-major) order. The embedding takes input_ids (seq, batch) and produces (hidden, seq, batch) after transpose. LlamaCausal outputs logits (vocab_size, seq, batch).

## Test Coverage

- **llama_config**: LlamaConfig validation and defaults
- **llama_mlp**: LlamaMLP forward + roundtrip + reference + tiled-vs-untiled
- **llama_attention**: LlamaAttention forward + roundtrip + reference + tiled-vs-untiled
- **llama_decoder**: LlamaDecoder forward + roundtrip + reference + tiled-vs-untiled
- **llama_model**: LlamaModel forward + roundtrip + reference + tiled-vs-untiled
- **llama_causal**: LlamaCausal forward + roundtrip + reference + tiled-vs-untiled

## Test Data (Safetensors)

Test data is generated by `generate_test_data.py` (in this directory) and saved as CTest fixtures. The script uses PyTorch + HuggingFace `transformers` to produce reference outputs. Each invocation covers a single block:

```bash
python tests/graph/model/llama/generate_test_data.py --block attention --output /tmp/data --seed 42
python tests/graph/model/llama/generate_test_data.py --block mlp --output /tmp/data --seed 42
python tests/graph/model/llama/generate_test_data.py --block decoder --output /tmp/data --seed 42
python tests/graph/model/llama/generate_test_data.py --block model --output /tmp/data --seed 42
python tests/graph/model/llama/generate_test_data.py --block causal --output /tmp/data --seed 42
```

Both NNTile and PyTorch are configured identically: RoPE is replaced with identity and causal masking is disabled so that outputs match exactly.

## Save/Load and HF Import/Export

Llama building blocks support:
- **save/load**: NNTile-native safetensors via `Module::save()` / `Module::load()`
- **import_hf/export_hf**: HuggingFace format conversion for `LlamaAttention`, `RMSNorm`, and submodules (`Linear`, `Embedding`, `GatedMlp`)

## Running Tests

```bash
ctest -R tests_graph_model_
```
